# Analysis method

1. Читать `docs/jira/STATUS_MODEL.md` и `docs/analytics/QUERY_CATALOG.md`.
2. Выполнять поиск только через MCP и дочитывать страницы результата.
3. Указывать JQL/CQL, момент выполнения, timezone, scope и определение метрики.
4. Разделять snapshot текущих полей и историю transitions.
5. Отделять факты из Atlassian от выводов агента.
