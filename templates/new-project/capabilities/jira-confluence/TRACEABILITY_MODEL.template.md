# Traceability model

| Jira entity | Required documentation | Link rule |
|---|---|---|
| Epic | Страница инициативы или roadmap | Jira link ↔ Page Properties |
| Story / Task | Требование, design или runbook — когда применимо | Jira key на странице и обратная ссылка |
| Bug | Исследование или postmortem для значимого инцидента | Двусторонняя ссылка |
| Release | Release notes и план внедрения | JQL/filter и связанные задачи |
| Process | Описание процесса и status update | Связанные проекты и JQL |

Не вести вручную копию всех связей в Markdown: фактические связи хранятся в Jira и Confluence.
