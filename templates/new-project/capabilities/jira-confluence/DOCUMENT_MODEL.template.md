# Document model

| ID | Русское название | Назначение |
|---|---|---|
| `requirement` | Требование | Что реализовать и как принять |
| `decision` | Архитектурное решение | Контекст, варианты и последствия |
| `research` | Исследование | Вопрос, источники и выводы |
| `meeting-notes` | Протокол встречи | Решения и действия |
| `runbook` | Операционная инструкция | Повторяемые действия |
| `postmortem` | Разбор инцидента | Причины и профилактика |
| `release-notes` | Описание релиза | Изменения и внедрение |
| `process-description` | Описание процесса | Роли, шаги и метрики |
| `status-update` | Статус-отчёт | Прогресс, риски и следующие шаги |

Страницы Confluence содержат: `Document type`, `Тип документа`, `Owner`, `Status`, `Related Jira issues`, `Last reviewed`, `Source of truth`.
