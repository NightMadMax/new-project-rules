# Security Policy

## Supported Versions

- Перечислите поддерживаемые версии или ветки.

## Reporting a Vulnerability

- Укажите приватный канал сообщения без публикации секретных контактов или
  credentials в репозитории.

## Response Expectations

- Подтверждение получения, первичная оценка и порядок раскрытия.
