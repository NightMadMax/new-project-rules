# Каталог шаблонов

Связанные шаблоны нового проекта:

- [[templates/new-project/README.template|README]] — описание и быстрый старт.
- [[templates/new-project/AGENTS.template|AGENTS]] — локальные правила агента.
- [[templates/new-project/INDEX.template|INDEX]] — карта заметок и файлов.
- [[templates/new-project/PROJECT.template|PROJECT]] — цели и границы проекта.
- [[templates/new-project/DOCS_INDEX.template|DOCS INDEX]] — индекс папки документации.
- [[templates/new-project/CHANGELOG.template|CHANGELOG]] — изменения и релизы.
- [[templates/new-project/ARCHITECTURE.template|ARCHITECTURE]] — текущее устройство системы.
- [[templates/new-project/ADR.template|ADR]] — отдельное значимое решение.
- [[templates/new-project/RESEARCH.template|RESEARCH]] — отдельное исследование.
- [[templates/new-project/CODE_REVIEW.template|CODE REVIEW]] — отдельное ревью.
- [[templates/new-project/TESTING.template|TESTING]] — стратегия качества.
- [[templates/new-project/ACTIONS.template|ACTIONS]] — действия вне git.
- [[templates/new-project/ENVIRONMENTS.template|ENVIRONMENTS]] — окружения.
- [[templates/new-project/RUNBOOK.template|RUNBOOK]] — повторяемая процедура.
- [[templates/new-project/POSTMORTEM.template|POSTMORTEM]] — разбор инцидента.
- [[templates/new-project/INTERFACES.template|INTERFACES]] — каталог интерфейсов.
- [[templates/new-project/DATA_MODEL.template|DATA MODEL]] — модель данных.
- [[templates/new-project/SECURITY.template|SECURITY]] — security policy.
- [[templates/new-project/THREAT_MODEL.template|THREAT MODEL]] — угрозы и меры.
- [[templates/new-project/INTEGRATIONS.template|INTEGRATIONS]] — внешние системы.
- [[templates/new-project/TOOLS.template|TOOLS]] — нестандартные инструменты.
