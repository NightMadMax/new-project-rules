Режим 1

Разбор входящих кандидатов

регулярно

Основная, частая работа: решить судьбу каждого присланного пользователем кандидата.

📁Где всё в этом режиме:Best Practices— открой здесь агента.
Разобрать кандидата и принять решение
Гдепапка Best Practices
СигналПользователь открыл PR с кандидатом; CI validate зелёный
Что сделать
  1. Открыть агента (Claude Code / Codex) в папке Best Practices.
  2. Написать: «Разбери кандидатов и прими решение» — или вызвать /review-practice-candidates.
  3. Пройти проверки: дедуп, качество, обобщённость, чистота, evidence — и вынести вердикт по каждому.
Итог✓ accept → практика в practices/<раздел>/PC-*.md (E1=trial, E2/E3=accepted), PR влит  ·  ✕ reject → причина в кандидате (журнал)
