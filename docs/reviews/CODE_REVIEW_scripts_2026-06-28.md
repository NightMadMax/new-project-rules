# Ревью скриптов от 2026-06-28

## Результат

Проверены shell- и PowerShell-версии bootstrap, настройки глобальных и scoped
правил, диагностика окружения и GitHub Actions. Все подтверждённые замечания
закрыты в версии `v1.9.0` и защищены регрессионными тестами.

| Приоритет | Проблема | Решение |
|---|---|---|
| Высокий | Shell-скрипты не находили шаблоны при запуске через `PATH`-symlink | Добавлено разрешение реального пути скрипта |
| Высокий | CI игнорировал parser errors PowerShell | Добавлен проверочный скрипт с ненулевым кодом при ошибках |
| Средний | `check-environment.ps1` падал без Git | Git-зависимые проверки защищены условием |
| Средний | PowerShell-bootstrap игнорировал ошибки `git init` и `git add` | Проверяется код каждой Git-операции |
| Средний | PowerShell-тест принимал ошибку `git status` за чистое дерево | Проверяется `$LASTEXITCODE` |
| Средний | CI маскировал сценарий без Git-идентичности | Идентичность изолирована внутри тестовых сценариев |
| Низкий | Использовались непереносимые или неоднозначные shell-конструкции | Убраны `ls -A` и лишние GNU-подобные аргументы, обработаны ошибки `grep` |

## Остаточный риск

GitHub Actions не запускает отдельный macOS runner. Перед публикацией изменений
shell-скриптов обязателен локальный прогон на macOS по инструкции
[[docs/quality/TESTING|TESTING]]. Windows PowerShell 5.1 и PowerShell 7
проверяются автоматически.
